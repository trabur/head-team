ht.turn()

a driver rotates the bottom of the steering wheel left ... every one shifts --> right (: a driver rotates the bottom of the steering wheel right ... every one shifts <-- left :)

simple turn by turn layering by only DECODING then ENCODING means two people in the same vehicle can navigate a distributed graph while also maintaining communication with one another sealed off from the outside world.

FIRST --> SECOND --> THIRD ... BEEP
let first = "1st Street" // road
let second = "2nd Street" // road
let third = "3rd Street" // road
let vehicle = ht.mobile()
let transportA = ht.lane(vehicle, first)
let transportB = ht.turn(transportA, second)
let transportC = ht.turn(transportB, third)
let hear = ht.listen(transportC)
let noise = ht.beep(transportC)

// mobile: enter...
// lane: transporting...
// lane: yield on SFM...
// turn: towards 2nd Street
// turn: towards 3nd Street
// listen: 3nd Street
// beep: HONK

many things start out as 2 which is why it's just exmaple code. meaning, we could have gone big and used an example of a bunch of people on a bus; or we could have gone local and used an example of 1-4 people in a car.

what is important to note is this ... with the ability TURN --> and the ability to HONK // we now have the ability to share secrets between one another using standard PASS & ACK protocols.
