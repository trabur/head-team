ht.beep()

JavaScript is a language that can be run from the browser. press developer tools (option + command + i) then open the console tab to start executing commands in order to follow along with this guide.

to begin, the BEEP/HONK method is good for checking system health status. when called, BEEP checks the database using PING/PONG commands and then responds back HONK if there is a successful connection:

let road = "Main Street"
let vehicle = ht.mobile()
let transport = ht.lane(vehicle, road)

> mobile: enter...
> lane: transporting...
> lane: yield on SFM...
> move: Main Street

let noise = ht.beep(transport)

> beep: HONK
